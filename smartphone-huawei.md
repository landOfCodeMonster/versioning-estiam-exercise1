# Smartphone XYZ

Description : Le smartphone XYZ est un appareil haut de gamme doté de nombreuses fonctionnalités avancées.

- Écran OLED de 6,5 pouces avec résolution 1080p

- Connectivité 5G
- Capteur d'empreintes digitales intégré
